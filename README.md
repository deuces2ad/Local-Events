# Local Events

Discover nearby events with a clean, scalable SwiftUI app. Browse local happenings, view details, bookmark your favorites, and get directions in Maps — all backed by a modular architecture with dependency injection for testability and growth.

A SwiftUI app with a modular architecture (`Core`, `Models`, `Networking`) and dependency injection via `DependencyContainer`. The app sets up `ContentView` with environment objects for shared dependencies and view model construction.

## Why this repository
Demonstrates a clean SwiftUI modular setup with dependency injection and testable layers suitable for scaling features like `EventListView`.
